## Directory Descriptions

- **app/**: Contains the main application code, including pages, layouts, and global styles.
- **components/**: Reusable UI components organized by feature or type.
- **hooks/**: Custom React hooks for shared logic across components.
- **lib/**: Core business logic and service libraries. Each subdirectory represents a major feature
- **trpc/**: tRPC API layer providing end-to-end type safety. Routers are organized by feature, and the context provides session and organization scoping.
- **schemas/**: Zod schemas for validating input data.
- **types/**: TypeScript type definitions for the project.
- **proxy.ts**: Middleware for handling API requests and responses.
