We've selected the best-in-class tools to provide a professional, scalable, and high-performance foundation for our project. Here's a breakdown of our tech stack:

- [Next.js](https://nextjs.org/docs): The React framework for the web, using the App Router for modern features like Server Components and Streaming.
- [Drizzle ORM](https://orm.drizzle.team/docs): A lightweight, high-performance TypeScript ORM for SQL databases.
- [Better Auth](https://better-auth.com/docs): A robust and flexible authentication library for Next.js.
- [tRPC](https://trpc.io/): For building end-to-end type-safe APIs without the boilerplate.
- [TanStack Query](https://tanstack.com/query/latest): For powerful data fetching, caching, and state management on the client.
- [Resend](https://resend.com/): For creating beautiful, responsive email templates.
- [React Email](https://react.email/): For creating beautiful, responsive email templates.
- [Shadcn UI](https://ui.shadcn.com/docs): Beautifully designed components built with Radix UI and Tailwind CSS.
